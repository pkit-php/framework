# Viewers do framework

- lista
  - `pkit/__layout`: corpo padrão das paginas do pkit
    - argumentos
      - redirect
      - lang
      - title
      - color
      - back-color
      - font-color
  - `pkit/code`: pagina que mostra um svg animado com base no código
    - argumentos
      - code
      - message
  - `pkit/loading`: pagina que mostra uma tela de carregamento animado
